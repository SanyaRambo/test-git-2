1. git status
2. git add [files] - добавляет файлы в stage
3. git commit -m "comment";
4. git log (Подробная инфа о записи(commit))/ git log --oneline  (Короткая инфа о записи(commit)) 
5. git push [rep_link] [branch_name]




Ещё команды
1. git reset (Добавили файл случайно, не хотели этого делат. Чтобы убрать его, используйте данную команду)
2. git diff (Просматривает все строки, которые изменяли или добавляли. Также если после дифф написать опр файл, то, собственно,дифф будет применяться только к этому файлу)
3. git reset --hard (Уберёт все изменения и вернёт то, что было до этого)
